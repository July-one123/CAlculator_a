ТЗ Калькулятора
Дата составление: 08.11.2025
Номер версии: 
Цель: создать корректно работающий калькулятор
Задачи: 
•	Создать шаблон
•	Проверка
•	Корректировка
Этапы: 
1.	Создать шаблон;
2.	Проверка работоспособности;
3.	Тест на ввод разных данных (0, отрицательные числа, числа с плавающей точкой, пустую строку, буквы, знаки, и их комбинации и знак(букву) для выхода)
4.	Исправление ошибок;
5.	Проверка, если работа программы исправна, то перейти дальше, иначе проверка.
Требованию к конечному продукту: корректно вычислял при вводе разных данных чисел.
Критерии оценки результата: 
•	проверяет верные ли данные ввели
•	не выводит ошибок
•	выводит верные данные
